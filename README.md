# javascript-beginner
This repo contains the javascript files and folders at a beginner level.

## Folder: day7-24-11-2025-express
To run the app install node_modules globally and run the index.js file using the command `node index.js`  

## Folder: day7-24-11-2025-todo-app
1. To run the app install node_modules globally and run the index.js file using the command `node index.js` 
2. To run the app locally change the program command in `./vscode/launch.json` to `"program": "${workspaceFolder}/day7-24-11-2025-todo-app/index.js"`
